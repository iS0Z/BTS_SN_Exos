# Stenographie_png_python3

Example of usage : 
Encoding a message in image
```bash
┌─[iS0Z]─[Stenographie_png_python3]
└──╼$python3 stenographie.py 
File : steganographie.py v0.1 starting 
 Done by https://github.com/iS0Z 
Use this script to add coded message in imgs

Use images that max size is 600px*600px

File will be saved as "s_[filename]" 

Choose an option

1) Encode | 2) Decode 

Choice : 1
Enter filename of image : orange.png
Tell me what to hide : 3.1415926
```
Decoding message in image : 
```bash
┌─[iS0Z]─[Stenographie_png_python3]
└──╼ $python3 stenographie.py 
File : steganographie.py v0.1 starting 
 Done by https://github.com/iS0Z 
Use this script to add coded message in imgs

Use images that max size is 600px*600px

File will be saved as "s_[filename]" 

Choose an option

1) Encode | 2) Decode 

Choice : 2
Filename : s_orange.png
The secret is : 3.1415926
```
